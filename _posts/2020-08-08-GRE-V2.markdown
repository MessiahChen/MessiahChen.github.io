# GRE阅读简介

长文章：4道题

中文章：3道题

短文章：2道题

逻辑题：1道题

| 题号  |   Section 1    |  题号  |   Section 2    |
| :---: | :------------: | :----: | :------------: |
|  1~6  |      填空      |  1~6   |      填空      |
| 7，8  |     短文章     |  7，8  |     短文章     |
| 9~12  |     长文章     |  9~11  |     中文章     |
| 13~16 | 填空（六选二） | 12~15  | 填空（六选二） |
|  17   |     逻辑题     |   16   |     逻辑题     |
| 18~20 |     中文章     | 17，18 |     短文章     |
|       |                | 19，20 |     短文章     |

建议10分钟做完10道填空题，20分钟做完10道阅读题。
